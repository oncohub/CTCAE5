CTCAE 5.0 PWA provides an easy way to browse Common Terminology Criteria for Adverse Events (CTCAE) 5.0, using Progressive Web App architectures. CTCAE 5.0 PWA is freely available for educational use.

  • Progressive: More functionality becoming available on subsequent visits.
  • Network Independent: Function both offline and online as well as on flaky connections.
  • Multi-platform support: Ensure consistent performance across multiple modern browsers and platforms.
  • Searchable: Show the search result with the matching text highlighted across terms, grades, and definitions.

Browser settings and search queries in Japanese may trigger to activate JCOG version automatically. Otherwise, activate manually.
