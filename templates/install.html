{{(shareData.langJa ? termlang : !termlang) ? 'Install' : 'インストール'}}

iOS / iPhone ( movie)
  1. Scan the barcode above or go to the website https://oncohub.github.io/ctcae5/.
  2. Tap the Share button .
  3. From the options, tap the 'Add to Homescreen'.
  4. You can notice an icon of 'CTCAE 5.0' added to your devices homescreen.
  5. Tap the icon from homescreen, then 'CTCAE 5.0' will be loaded.
Android
  1. Scan the barcode above or go to the website https://oncohub.github.io/ctcae5/.
  2. You can see a Add to Home screen prompt from the bottom of screen.
  3. If not, tap the menu button and then select 'Homescreen' option.
  4. You can notice an icon of 'CTCAE 5.0' added to your devices homescreen.
  5. Tap the icon from homescreen, then 'CTCAE 5.0' will be loaded.
MacOS - Chrome
  1. Go to the website https://oncohub.github.io/ctcae5/.
  2. Reload the page.
  3. Tap the Options button and select the Install CTCAE 5.0.
  4. Then, you can see a new app in the 'user_name/Application/Chrome Apps' folder.
MacOS - Safari
  1. Go to the website https://oncohub.github.io/ctcae5/.
  2. Add https://oncohub.github.io/ctcae5/ to bookmark. [NOTE] Please make sure the URL (not containing 'index.html', '#/tab/home' or other strings).
  3. Launch from the bookmark.
Windows 10
  1. Go to the website https://oncohub.github.io/ctcae5/.
  2. Tap the more options button.
  3. Tap the Pin to Start option, then you can notice an icon of 'CTCAE 5.0' added to your Start Menu.
  4. Tap the icon from the Start Menu, then 'CTCAE 5.0' will be loaded.
iOS / iPhone (ムービーを)
  1. 上記のバーコードをスキャンするかブラウザに直接 https://oncohub.github.io/ctcae5/ を入力し、ウェブページを表示する。
  2. シェアボタン をタップする。
  3. オプションから「ホーム画面へ追加」を選択肢する。
  4. ホーム画面に作成された「CTCAE 5.0」アイコンを見つけ、タップする。
Android
  1. 上記のバーコードをスキャンするかブラウザに直接 https://oncohub.github.io/ctcae5/ を入力し、ウェブページを表示する。
  2. 「ホーム画面へ追加」プロンプトをタップする。
  3. プロンプトが出ない場合には、メニューポタから、「ホーム画面へ追加」を選択する。
  4. ホーム画面に作成された「CTCAE 5.0」アイコンを見つけ、タップする。
MacOS - Chrome
  1. ブラウザに直接 https://oncohub.github.io/ctcae5/ を入力し、ウェブページを表示する。
  2. ページをリロードする。
  3. オプションより「CTCAE 5.0をインストールする」を選択する。
  4. [ユーザ]/アプリケーション/Chrome アプリ フォルダに作成された「CTCAE 5.0」アイコンを見つけ、クリックする。
MacOS - Safari
  1. ブラウザに直接 https://oncohub.github.io/ctcae5/ を入力し、ウェブページを表示する。
  2. ブックマークに https://oncohub.github.io/ctcae5/ を追加する。 [注意] URLに上記以外の文字（例えば、'index.html'、'#/tab/home'など）を含まないようにする。
  3. 登録したブックマークを選択する。
Windows 10
  1. ブラウザに直接 https://oncohub.github.io/ctcae5/ を入力し、ウェブページを表示する。
  2. オプションより、「スタートにピン留めする」を選択する。
  3. スタートメニューに作成された「CTCAE 5.0」アイコンを見つけ、タップする。

{{(shareData.langJa ? termlang : !termlang) ? 'Uninstall' : 'アンインストール'}}

Unregister the Service Worker and remove cache

iOS (iPhone)
  1. Go to the Settings and tap on Safari.
  2. Tap the Remove All Website Data.
Android
  1. Open Settings and select Storage.
  2. Tap the Other Apps entry.
  3. Choose an app.
  4. On the app’s info page, tap the Clear Cache option.
MacOS - Chrome
  1. Tap the Options button and select the Uninstall CTCAE 5.0.
MacOS - Safari
  1. In the Safari menu select Preferences.
  2. In the Advanced section, toggle the Show Develop menu to activate.
  3. Click on the Develop in the menu bar and select the Empty Caches.
  4. Close the Safari browser.
Windows 10
  1. Click the icon with three horizontal dots in the top right of the window to open the More menu.
  2. Click the button that says “Choose what to clear”.
  3. Toggle the check boxes to delete.
  4. Then clear.

Service Worker の登録を解除し、キャッシュを削除する。

iOS (iPhone)
  1. 「設定」 > 「Safari」を選ぶ。
  2. 「履歴とWebサイトデータを消去」する。
Android
  1. 「設定」 > 「アプリと通知」を選ぶ。
  2. 「〇〇個のアプリをすべて表示」する。
  3. 「CTCAE 5.0」を選ぶ。
  4. 「ストレージ」 > 「キャッシュを削除」を選ぶ。
MacOS - Chrome
  1. オプションから、「CTCAE 5.0をアンインストール」する。
MacOS - Safari
  1. Safariのメニューの「環境設定」を選ぶ。
  2. 詳細タブから「メニューバーに"開発"メニューを表示」を選ぶ。
  3. メニューバーの「開発」から「キャッシュを空にする」を選ぶ
Windows 10
  1. オプションから「設定」を選ぶ。
  2. 「クリアするデータの選択」する。
  3. 「Cookieと保存済みのWebサイトデータ」と「キャッシュされたデータとファイル」をチェックし、「クリア」する。
